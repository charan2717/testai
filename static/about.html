About Our AI Chat

Welcome to our AI-powered chatbot named Base! This project is designed to provide a smooth, intelligent, and interactive chat experience.

üåç How Does Our API Work?

We use a simple yet powerful approach to generate responses. The backend is built using Flask, and we connect to an AI model API to handle user queries.

üõ† API Workflow:

When a user sends a message, our backend:

  1. Receives the message via a Flask API endpoint.
  2. Sends the message to an AI model (e.g., meta-llama).
  3. Processes the response and sends it back to the frontend.
  4. Once you refresh the page the chat history will be gone.

üöÄ Example API Call:

import requests

API_KEY = "YOUR_API_KEY"
API_URL = "https://api.together.xyz/v1/chat/completions"
HEADERS = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
DATA = {
    "model": "meta-llama/Llama-3.3-70B-Instruct-Turbo",
    "messages": [{"role": "user", "content": "Hello, Base!"}]
}

response = requests.post(API_URL, headers=HEADERS, json=DATA)
print(response.json())
